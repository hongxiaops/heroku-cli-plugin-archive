# Usage
<!-- usage -->
```sh-session
$ npm install -g heroku-cli-plugin
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
heroku-cli-plugin/1.0.0 darwin-x64 node-v14.5.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
